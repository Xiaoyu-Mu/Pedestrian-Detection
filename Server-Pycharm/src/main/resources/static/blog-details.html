With the development of information technology and the progress of society, we need continually extract feature information from image. Traditional method includes manual handling, and model detection based on machine learning. Although the method of manual processing is of high accuracy, it take too long and require too many human resources. Although it was upgraded to model detection based on machine learning and the processing speed was significantly improved, the accuracy of feature extraction was obviously lower than that of manual. Obviously, traditional methods face many kinds of problems, like heavy workload, inefficiency and the processing accuracy is low. Besides, traditional methods cannot solve mass data. Therefore, it is necessary to work on a high-technology model. Object detection based on deep learning is the key technology to solve the above disadvantages. After certain data training, it can be comparable to manual processing in accuracy, and the processing speed is much faster than the detection based on machine learning, and the convolution neural network makes the system more stable and efficient. Therefore, the object detection based on deep learning has great research significance.

There are many object detection techniques based on deep learning. The traditional object detection often mark the object with rectangular frame in the image. First, most successful target detectors enumerate potential target locations, then locations are classified. However, object detectors are not able to perceive internal information, so that they cannot make full use of the internal features of an object. Accordingly, this kind of method lead to false detection and waste a lot of time. At present, the more advanced technology is to set the target as a point (the central point of bbox) when we are building the model. detectors find the central point by key points, and regress to other attributes, such as size, location, orientation, etc. Key points include the central point, the upper left corner point, and the lower right corner point. These three points enable the network to have the ability of sensing the internal information of the object at a small cost, and thus can effectively suppress false detection

Object detection based on deep learning started earlier in other countries. Studies have been carried out before 2012, and the main contents is task partitioning. However, the results were not good. After 2012, object detection is roughly divided into two categories, requiring candidate boxes and not requiring candidate boxes, which increases data, that is, the training time is longer, and the test generalization performance is better. At present, this technology has been applied to transportation, biology, medical and other industries.

After learning a series of knowledge and related literature, I decided to use client-server mode as the architecture of the overall system. The main function of the server is to provide identification services, and recognition services are mainly object recognition methods based on deep learning. The main work of the server is to receive the client data, extract the feature information and process the image data, and finally transfer the marked image to the client. The client uses JavaWeb technology to send HTTP requests to the server, then the server returns the results to the client in JSON format. The system is divided into two large modules, which are offline network training and online object recognition. These two large modules also include some small modules.

1. Accuracy of feature extraction: Convolutional neural network is used to extract features and increase the number of layers in dense blocks. By using complex synthesis functions, convolution kernels of different sizes, pooling kernels, and compression model to improve the network, which improves the accuracy of the system and processing speed.

2. System performance problems: In terms of algorithm, the model is constructed based on deep learning. In mathematics, a large number of matrix operations and linear operations are used. In hardware, CPU is used for computing acceleration.

technical feasibility: Nowadays, artificial intelligence technology develops rapidly and gradually matures. In the research field, there are more and more deep learning and convolutional neural network research results. The level of literature has also gradually improved. It provides great convenience for future generations to learn. In terms of hardware, GPU and CPU performance are getting higher and higher, providing strong hardware support for theoretical implementation.

economic feasibility: The cost of server performance is manageable and does not require a large number of people to collaborate on development. It can be regarded as a personal project, so the labor cost is limited to individuals. The completed system is relatively stable and the maintenance cost is low, so it is economically feasible.

Social feasibility: The industry 4.0 era is fully opened, and the field of computer vision is also experiencing rapid development. Social production led to a clear division of labor, but also prompted a strong demand for machines to replace labor. Machines can calculate at speeds that humans can't, Algorithms can catch information that the human eye can't. In addition, the benefits brought by the new detection model are far greater than the traditional detection benefits.
