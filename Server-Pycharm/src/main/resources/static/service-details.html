Object detection based on YOLOv3

The purposes of zooming out images (or Subsampled): 1. Make the image conform to the size of the display area; 2. Generate a thumbnail of an image.

The purposes of zooming in images (or upsampling): 1. zoom in the original image, so it can be displayed on higher resolution display devices.

Anchor box: Help us set the width and height of the detection target. When the model makes a prediction, it can take advantage of this already defined width and height to predict. The function is to assist in processing x_offset, y_offset, h and w.

Confidence:predict the IOU value of the prior box and ground truth box. For example, an image includes houses, trees, cars, person and so on, but there is no face in this image when the model is recognizing faces, then the confidence is 0. The more faces in the box, the greater the confidence (probability of having an object), so confidence is a very important parameter in detection.

1. Yolov3 extracts multiple feature layers for target detection. Three feature layers were extracted, and they are located in the middle, lower, and bottom layer of the feature extraction network darknet53. The shape of the three feature layers is (52,52,256), (26,26,512), and (13,13,1024). These three feature layers are then used for concat with other feature layers after upsampling.

2. The third feature layer (13,13,1024) is convolution processed for 5 times (for feature extraction). After processing, one part is used for convolution and UpSampling, and the other part is used for output corresponding prediction results (13,13,75). Conv2D 3×3 and Conv2D 1×1 convolved to adjust the channel to the desired size of the output.

3. The feature layer of (26,26,256) is obtained after convolution and upsampling. It is then concat with the feature layer (26,26,512) in the Darknet53 network, and the shape is (26,26,768). After 5 convolution, part is used for convolution up-sampling, and the other part is used for output corresponding prediction results (26,26,75). Conv2D 3×3 and Conv2D1×1 are adjusted as above for channels.

4. Then concat the feature layer of convolution + upsampling in 3 with the feature layer of shape (52,52,256), then convolution is performed to obtain the feature layer whose shape is (52,52,128). Finally, convolve Conv2D 3×3 with Conv2D 1×1 to obtain the feature layer (52,52,75).
